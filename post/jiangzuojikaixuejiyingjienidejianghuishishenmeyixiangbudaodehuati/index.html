讲述
我们的艺术与生活


艺术源于生活而又高于生活，不管是绘画、摄影抑或语言，艺术家们通过各种方式来表达人们内心的想法，在新学期伊始，让我们通过这几场有关艺术的讲座来丰富自己对待生命的看法。




浅说新闻漫画


= 张耀宁 =

时间：2016年9月10日上午9:30


地点：省图长江报告厅


主讲人介绍：毕业于北京教育学院美术系，《中国日报》高级编辑，作品多次入选全国性漫画展览及综合性美展。



摄影如何改变了我们的生活
= 王保国 =

时间：2016年9月10日14：30—16：30


地点：武汉市图书馆


主讲人介绍：中国艺术研究院摄影所研究员、《中国摄影家》杂志副主编。著有《谈影唯缺颂红妆——摄影阅读笔记》和《东方照相记》。



让摄影成为生活方式
= 张晓冬 =

“梦－－梦境、梦域、梦幻”张晓冬博士个人摄影艺术作品展预告


展览时间：2016年9月5日—10月7日9：00-16：30，16：00停止入馆；逢周一闭馆，周末、节假日正常开放。


地点：武汉大学万林艺术博物馆


作者介绍：武汉大学法学博士；摄影家、晓冬视觉艺术工作室创办人、梦画廊创办人；中国摄影家协会会员，英国皇家摄影协会会员，香港摄影学会会员；CFP视觉中国签约摄影师，东方IC签约摄影师，中国网图片中心签约摄影师，全球网签约摄影师。




雪国小城这两年

= 刘慧 =

时间：2016年9月10日10：00－11：30


地点：武汉市图书馆


主讲人介绍：武汉纺织大学外语学院副教授。主讲大学英语、美国戏剧等课程。多次参加国际学术会议，在《外国文学研究》等刊物上发表论文20余篇。2013-2015公派至俄罗斯莫尔多瓦国立大学外国语学院任教，为武汉纺织大学与莫尔多瓦大学共同申请设立莫尔多瓦大学孔子学院做前期的探索性工作。
